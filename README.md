# Samba Presentation

Présentation interactive sur Samba - le protocole de partage de fichiers et d'authentification Linux.

**Accédez à la présentation :** https://stephdl.github.io/samba_presentation/

## À propos

Cette présentation couvre :
- Les origines de Samba et son évolution depuis 1992
- L'architecture et les composants principaux
- Les modes de fonctionnement (Workgroup, Domain Member, Active Directory DC)
- L'authentification (NTLM vs Kerberos)
- Les annuaires et le contrôle d'accès
- Les cas d'usage et avantages/limitations
- Les développements actuels (2024-2025)

**Présentateur :** Stéphane de Labrusse
